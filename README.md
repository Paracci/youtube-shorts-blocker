# YouTube Shorts Channel Blocker & Ad Skipper

This extension has been developed to improve your experience while watching YouTube Shorts. You can instantly block unwanted channels, skip ads automatically, and enjoy the highest video quality possible!

## Features

- 🚫 **One-Click Channel Blocking:** Add unwanted channels to the "Do not recommend" list with a single click using our custom button that seamlessly blends with the Shorts interface.
- ⏭️ **Auto Ad Skipper:** Instantly mutes and skips sponsored/ad videos that appear in your Shorts feed.
- 📺 **Force Highest Quality (4K/HD):** Prevents YouTube from lowering the video quality based on your connection speed, forcing the video to play at its highest available resolution (up to 2160p).
- 🎨 **Cleaner Interface:** The native YouTube "Remix" button is intentionally hidden so that our "Block" button perfectly aligns and preserves a clean button layout.
- ✨ **Native YouTube Design:** The added block button perfectly matches YouTube's native button style, spacing, and typography.

## Manual Installation (Developer Mode)

To install this extension on Chromium-based browsers like Google Chrome, Edge, Brave, or Opera outside the web store, follow these steps:

1. Save/extract this folder to a safe location on your computer.
2. Type `chrome://extensions/` (or `edge://extensions/`) in your browser's address bar and hit Enter.
3. Enable **"Developer mode"** in the top right corner.
4. Click the **"Load unpacked"** button in the top left.
5. Select the folder containing this extension (the folder with `manifest.json` inside).
6. The extension is now installed! Go to the YouTube Shorts page to test it out.

## Notes
- To add a custom icon to your extension bar, simply place your own images named `icon16.png`, `icon48.png`, and `icon128.png` in the extension folder. (If not provided, Chrome will use a default puzzle piece icon - the extension will still work perfectly).
- The extension relies on YouTube's current DOM structure. If it stops working suddenly, it's highly likely YouTube has updated their site's layout.
